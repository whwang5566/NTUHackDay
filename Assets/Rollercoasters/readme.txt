This package contains 2 different types of an detailed animated rollercoaster for background decoration.
Just drag the prefabs into your scene and arrange them as you like.

You may also want to have a look at the example scene that contains all the prefabs.

Textures-sizes between 512x512 to 2048x2048.
Textures are created using Ambient-Occlusion.
Both Coasters using 3 keyframe-animated trains (not physically correct).
Each Coaster uses about 80.000 Polygons.

The original FBX-file is also included in the package.
